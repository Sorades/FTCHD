# FTCHD
An interpretable deep learning model for first-trimester fetal cardiac screening.

## Env
Recommended environment:
- python 3.12.4
- pytorch 2.4.0
- torchvision 0.19.0
- lightning 2.4.0
- transformers 4.52.3

See `pyproject.toml` for more details. If you are using pixi, simply run:
```bash
pixi i && pixi shell
```
